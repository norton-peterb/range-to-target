# range-to-target

## Introduction

Project to interface Ultrasonic Sensor with LCD Display

## Wiring

### LCD Display (TM1637)

|Pin ID|Purpose|Arduino Pin|
|-----|-----|-----|
|GND|Ground Power|GND|
|VCC|5V Power Source|5V|
|DIO|Digital I/O|4|
|CLK|Digital Clock|5|

No Additional Circuit components are required

### Ultrasonic Module (HC-SR04)

|Pin ID|Purpose|Arduino Pin|
|-----|-----|-----|
|GND|Ground Power|GND|
|VCC|5V Power Source|5V|
|TRIG|Trigger Pin|9|
|ECHO|Echo Pin|8|

No Additional Circuit components are required
